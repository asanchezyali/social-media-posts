Matrices y coordenadas — cómo una base se vuelve un sistema de coordenadas. 🧭

Parte de la serie Matemáticas para Machine Learning. Elegir una base ordenada es elegir un sistema de coordenadas: los números que usamos para describir un vector dependen de la base. Y una aplicación lineal, en coordenadas, no es más que multiplicar por una matriz.

En este resumen:
• Coordenadas: fijada una base ordenada B, todo vector se escribe de una sola forma, x = α₁b₁ + … + αₙbₙ. Esos α son su vector de coordenadas.
• Una base = un sistema de coordenadas: el mismo vector tiene coordenadas distintas en bases distintas (lo ves con la base estándar vs. una base rotada).
• La matriz de transformación A_Φ: en su columna j van las coordenadas de Φ(bⱼ) en la base de llegada. Entonces ŷ = A_Φ x̂.
• Matrices que transforman: sobre un cuadrado de puntos, cada matriz es una transformación — rotar 45°, estirar ×2, o una general (reflexión + rotación + estiramiento).
