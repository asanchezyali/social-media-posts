El truco del −1 — leer una base del núcleo de un vistazo. 🧩

Parte de la serie Matemáticas para Machine Learning. Un atajo precioso: cuando tienes una matriz en forma reducida (RREF), puedes leer una base del núcleo de Ax = 0 casi sin cuentas.

En este resumen:
• Por qué el núcleo (kernel / null space) importa.
• Se parte de la forma escalonada reducida, sin filas nulas.
• Rellenas la diagonal insertando filas [0 … −1 … 0] donde falta el pivote.
• Las columnas que llevan un −1 en la diagonal son, directamente, una base del núcleo.
