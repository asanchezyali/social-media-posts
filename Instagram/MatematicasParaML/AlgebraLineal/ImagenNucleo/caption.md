Imagen y núcleo — qué llega a cero y qué se puede alcanzar. 🎯

Parte de la serie Matemáticas para Machine Learning. Toda aplicación lineal Φ: V → W carga consigo dos subespacios: el núcleo (lo que se aplasta a cero) y la imagen (lo que de verdad se alcanza).

En este resumen:
• El núcleo (o espacio nulo) ker(Φ) = {v : Φ(v) = 0} y la imagen (o rango) Im(Φ) = Φ(V).
• El mapa completo: todo el núcleo colapsa al 0, y Φ es inyectiva exactamente cuando ker(Φ) = {0}.
• Con una matriz Φ(x) = Ax: la imagen es el espacio columna (span de las columnas), rk(A) = dim(Im Φ); el núcleo es la solución de Ax = 0 y codifica las relaciones entre columnas.
• Un ejemplo completo: forma reducida por filas y base del núcleo.
• El teorema del rango y la nulidad (fundamental de las aplicaciones lineales): dim(ker Φ) + dim(Im Φ) = dim(V). Y si dim(V) = dim(W), inyectiva, sobreyectiva y biyectiva son lo mismo.
