Base y dimensión — el esqueleto mínimo de un espacio vectorial. 🦴

Parte de la serie Matemáticas para Machine Learning. Una base es el conjunto de vectores justo necesario para construir todo el espacio: ni le sobra ni le falta. Y su tamaño es la dimensión.

En este resumen:
• Conjunto generador y span: cuando todo vector es combinación de unos pocos.
• Qué es una base: el generador mínimo (equivalente: un generador linealmente independiente).
• Las tres caras equivalentes de una base: generador mínimo, independiente máximo, y representación única de cada vector.
• La dimensión = número de vectores de una base = número de direcciones independientes (¡ojo, no es el número de componentes!).
• Cómo hallar una base: pon los vectores como columnas, lleva la matriz a forma escalonada y quédate con las columnas pivote.
