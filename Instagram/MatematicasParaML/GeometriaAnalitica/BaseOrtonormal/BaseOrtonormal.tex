% -------------------------------------------------- qué es -------
\topic{¿Qué es?}
\hand{Una \uhand{base ortonormal} (ONB) es una base cuyos vectores son \hl{perpendiculares} entre sí y de longitud $1$:}
\keyeq{\langle\mathbf{b}_i,\mathbf{b}_j\rangle = \begin{cases}1 & i=j\\ 0 & i\neq j\end{cases}}
\obs{La base estándar $(\mathbf{e}_1,\dots,\mathbf{e}_n)$ es ortonormal con el producto punto. Si solo son perpendiculares (sin normalizar), es \hl{ortogonal}.}

% -------------------------------------------------- por qué -----
\topic{¿Por qué es tan cómoda?}
\hand{Con una ONB, las coordenadas de un vector se leen con \hl{productos internos} —sin resolver ningún sistema:}
\keyeq{\mathbf{x} = \textstyle\sum_{i} \langle\mathbf{x},\mathbf{b}_i\rangle\,\mathbf{b}_i}
\obs{Cada coeficiente $\langle\mathbf{x},\mathbf{b}_i\rangle$ es la \hl{proyección} de $\mathbf{x}$ sobre esa dirección.}

% -------------------------------------------------- ilustración -
\topic{Míralo}
\hand{En $\mathbb{R}^2$, $\mathbf{b}_1\perp\mathbf{b}_2$ y cada uno mide $1$. Descomponer $\mathbf{x}$ es soltar perpendiculares:}
\begin{center}
\begin{tikzpicture}[x=11mm,y=11mm,>={Latex[length=2mm]}]

  \coordinate (X) at (1.7,1.2);

% -------------------------------------------------- Gram-Schmidt -
\topic{¿Cómo conseguirla?}
\obs{¿No tienes una ONB? \hl{Gram}--Schmidt convierte cualquier base en una: resta proyecciones para ortogonalizar y luego normaliza cada vector.}
