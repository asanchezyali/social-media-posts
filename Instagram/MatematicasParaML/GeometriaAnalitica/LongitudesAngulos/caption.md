Longitudes y ángulos — el producto interno se vuelve geometría. 📐

Tercer post del módulo de Geometría Analítica, dentro de la serie Matemáticas para Machine Learning. Una vez que tienes un producto interno, aparece toda la geometría: longitudes, distancias, ángulos y ortogonalidad.

En este resumen:
• La longitud: todo producto interno induce una norma, ‖x‖ = √⟨x,x⟩. Y siempre vale Cauchy–Schwarz: |⟨x,y⟩| ≤ ‖x‖‖y‖.
• La distancia: d(x,y) = ‖x − y‖. Es una métrica (no negativa, simétrica, desigualdad triangular). Con el producto punto, la distancia euclidiana de siempre.
• El ángulo: gracias a Cauchy–Schwarz, cos ω = ⟨x,y⟩ / (‖x‖‖y‖) siempre está en [−1,1], así que define un ángulo real entre dos vectores.
• Ortogonalidad: x ⊥ y cuando ⟨x,y⟩ = 0 (ángulo recto). ¡Ojo! depende del producto interno elegido.
• Matrices ortogonales: columnas ortonormales ⟹ AᵀA = I ⟹ A⁻¹ = Aᵀ. Preservan longitudes y ángulos: son las rotaciones y reflexiones.
